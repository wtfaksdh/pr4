Набор бариста "Barista Kit"

Нажмите на фото для увеличения

Полный набор профессиональных аксессуаров для приготовления кофе.

Описание

Набор бариста "Barista Kit" включает всё необходимое для приготовления и украшения кофейных напитков. Аксессуары из нержавеющей стали и натурального дерева упакованы в стильный деревянный бокс.

Характеристики

  • Темпер 58 мм из нержавеющей стали
  • Распределитель кофе с регулировкой
  • Молочный питчер 350 мл
  • Клинер для форсунок
  • Деревянная подставка
  • Деревянный бокс для хранения
